# Q&A：split と match の使い分け

## Q1. split() の区切り文字って何でもいいの？

### 疑問
`'a,b,c'.split(',')` の補足に「区切り文字は残らない」とあるけど、`,` が特別なの？

### 答え
`,` は特別ではなく、**「どこで切るか」を指定するだけ**なので何でもOKです。

```javascript
'a,b,c'.split(',')   // → ['a', 'b', 'c']
'a+b+c'.split('+')   // → ['a', 'b', 'c']
'a-b-c'.split('-')   // → ['a', 'b', 'c']
'a b c'.split(' ')   // → ['a', 'b', 'c']
'a|b|c'.split('|')   // → ['a', 'b', 'c']
```

`split()` の引数に渡した文字が「切り取り線」になるイメージです。
切り取り線自体は結果に残りません。

## Q2. 区切り文字がない文字列を分割したい

### 疑問
`'aabbcc'` のように区切り文字がない文字列を `['aa', 'bb', 'cc']` にしたい。どうすればいい？

### 答え
`split()` は区切り文字が必要なので使えません。代わりに `match()` を使います。

```javascript
'aabbcc'.match(/.{2}/g)
// → ['aa', 'bb', 'cc']
```

### `/.{2}/g` の読み方（正規表現）

| 部分 | 意味 |
|---|---|
| `/` `/` | 正規表現の区切り記号（クォートの代わり） |
| `.` | 任意の1文字（どんな文字でもOK） |
| `{2}` | 直前のパターンを2回繰り返す（= 2文字分） |
| `g` | 文字列全体を繰り返し探す（global フラグ） |

`g` フラグを付けないと最初の1件しか取れないので注意。

```javascript
'aabbcc'.match(/.{2}/)   // → ['aa']  ← 最初の1件だけ
'aabbcc'.match(/.{2}/g)  // → ['aa', 'bb', 'cc']  ← 全件
```

何文字ずつに分割するかは `{n}` の数字を変えるだけです。

```javascript
'aaabbbccc'.match(/.{3}/g)  // → ['aaa', 'bbb', 'ccc']
```

---

## まとめ：split と match の使い分け

| メソッド | 使いどころ |
|---|---|
| `split('区切り文字')` | 特定の文字を目印に切り分けたいとき |
| `match(/パターン/g)` | 「何文字ずつ」など、パターンで取り出したいとき |
